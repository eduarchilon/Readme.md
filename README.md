## Getting Started


### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/gm2dev/beer-logger.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install SASS
   ```sh
   npm install -g sass
   ```
4. Install Prettier
   ```sh
   npm install --save-dev --save-exact prettier
   ```
